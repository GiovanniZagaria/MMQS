Requirements

  • Install IBM Rational.
  • Local copy of the UMLsec profile.

Creating a model

  1. Create a new model project: File → New → Model Project
  2. Enter a name for your project.




  3. Click Next twice.
  4. Ensure that the package type is set to Model and choose your Default diagram type.




  5. Click Finish.

Importing the UMLsec profile

  1. Select the root element of your model in the project explorer.
  2. In the properties view click on Profiles and add a new profile.


  3. In the appearing Select Profile window choose File and browse to your local copy of the UMLsec.profile.uml file.
  4. Click on Open and confirm your decision by clicking OK.
  5. Save your project. You are now ready to start modeling with UMLsec.
