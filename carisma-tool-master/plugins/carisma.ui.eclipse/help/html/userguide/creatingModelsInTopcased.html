Requirements

  • Download Topcased and unpack it.
  • Install CARiSMA in Topcased.

Creating a model

  1. Create a new TOPCASED Project: File → New → Topcased → Topcased Project
  2. Afterwards create a new UML model with TOPCASED: File → New → Topcased → Topcased Diagrams → UML Model with TOPCASED
  3. In the appearing window choose a directory, enter a name for your new model and select the type of the initial diagram.




  4. Click Finish.

Importing the UMLsec profile

  1. If the model is open, close the *.umldi file of your project. Open the corresponding *.uml file in the Project Explorer.
  2. Expand the treeview in the Model Explorer and select the <Model> element.
  3. Select the menu item UML Editor → Package → Apply Profile.
  4. Select the UMLsec profile in the appearing window.




  5. Click Add to apply the profile and OK to confirm your decision.
  6. Save your project.
  7. Close the *.uml file of your project and open the *.umldi file. UMLsec can now be applied to model elements.




Tips

To display the stereotypes and their tag values in the modeler, click Window → Preferences in the menu bar. In the appearing window select Topcased → Editors → UML/sysUML → All Diagram → Stereotype and enable Display stereotype attribute as a note.
