First Steps

In this guide you will need the following components:
  • An OCL Library Model
  • An Extension Model
Therefore follow the the creation guides of those models.

Extend the BPMN2 Model

If you want to extend your standard BPMN2 model with additional information, it is necessary to link up the BPMN2 model with the created Extension model. Therefore create a documentation node under the definitions root with the following context:

<<ext=yourModelName.bpmn2extension>>

The Extension Model must be saved into the same directory as the BPMN2 model. Keep in mind that the extension is optional.
To select the desired OCL-Constraints add a documentation node as well and use the following context:

<<pattern={Var1, Var2}>>

It is possible to select the OCL-Constraints by name or index. The wildcardoperator '*' is also supported by <<pattern={*}>>
