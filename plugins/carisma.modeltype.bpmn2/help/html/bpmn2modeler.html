Requirements

  • Eclipse 3.6 (Helios) or newer
  • Graphiti framework. (Update site: http://download.eclipse.org/graphiti/updates/0.7.1/)
  • Eclipse Modeling Framework

Installation

For a quick installation overview go to the video section.
There are two possibilities to get the latest editor version:

Using update site

Use http://download.eclipse.org/bpmn2-modeler/site update site update site.
Note: The Feature project includes the bpmn2 meta model, so select it if you have not currently installed the bpmn2 meta model.

Using GIT

If you want to run the editor from source you also need to check out the source project of the bpmn2 meta model at http://git.eclipse.org/c/bpmn2/.
Use http://git.eclipse.org/c/bpmn2-modeler/org.eclipse.bpmn2-modeler.git to get the source version of the editor.
