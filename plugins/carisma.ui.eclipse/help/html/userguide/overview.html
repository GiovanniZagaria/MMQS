Tool overview

CARiSMA is not a single tool but a set of Eclipse plugins. The basis of the CARiSMA tool is Eclipse and its EMF framework. The core tool is contained in the plugin carisma.core. It provides the GUI components, loading mechanism, a check registry, and general utility classes.

The core tool is extended by a model type specific component, which is carisma.modeltype.uml2 in case of UML2. This component provides the definition of the meta model, a concrete loader, and convenience methods and utility functions for working with models of this type. The model type components present in CARiSMA use standard component providing the meta model. In the case of UML2 the plugin uses the UML2 Plugin for Eclipse. Using the model type plugin and the core plugin, plugins for model analysis can be developed, e.g. the carisma.check.staticcheck plugin.

The analysis plugins for UML models may depend on UML profiles which are provided by profile plugins such as carisma.profile.umlsec. This plugin, for instance, contains the UMLsec profile which is used by most UMLsec analysis plugins (i.e. those migrated from the old UMLsec tool).
